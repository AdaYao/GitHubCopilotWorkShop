# COVID19全球检测数据

## 简介

        这个数据集 tested_worldwide.csv 来自 Kaggle。这个数据集记录了随时间进行的检测次数，对于帮助理解每天报告的病例数以及理解COVID-19在每个国家的真实传播情况非常重要。
